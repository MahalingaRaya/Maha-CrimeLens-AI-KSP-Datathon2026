## 📌 The Challenge
The Karnataka State Police (KSP) manages a continuously expanding repository of crime data across 1,100+ police stations. Currently, analyzing this fragmented data for proactive policing is challenging. 

**Maha CrimeLens AI** directly addresses **Datathon 2026 - Challenge 02** by transforming raw FIR records into actionable intelligence through predictive modeling, hotspot clustering, and visual geospatial analytics.

## 🚀 Key Features
*   **Geospatial Risk Radar:** Visual hotspot detection using simulated spatial clustering (DBSCAN principles) to identify high-risk districts.
*   **Predictive AI Scoring:** AI-driven severity analysis prioritizing active alerts for rapid resource deployment.
*   **Real-time Analytics Dashboard:** Interactive, dark-mode visualization interface tracking district-level crime trends and FIR processing.
*   **Mobile-First Architecture:** Engineered for speed and accessibility across devices for officers on the field.

## 🧠 System Architecture (Prototype Phase)
To demonstrate the intelligence pipeline rapidly, this prototype utilizes a specialized JavaScript data-engine inside React. 

*   **Frontend:** React.js, TypeScript, Lucide Icons, Recharts (for data visualization).
*   **Data Engine:** Mock KSP Database simulating robust PostgreSQL/PostGIS spatial queries and link-analysis for demonstration purposes.

## 💡 Why "CrimeLens"?
Law enforcement doesn't just need to see data; they need to *understand* it. This platform acts as an intelligent lens, filtering the noise to show KSP command centers exactly where to deploy resources next.
